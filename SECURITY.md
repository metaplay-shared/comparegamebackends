# Security Policy

[comparegamebackends.com](https://comparegamebackends.com) is a static, informational website. It has no user accounts, no login, and no server-side data store, so its security surface is small. We still want to hear about any concern you find.

## Reporting a vulnerability

Please report security concerns privately through GitHub's built-in vulnerability reporting rather than opening a public issue:

1. Go to the **Security** tab of the [repository](https://github.com/metaplay-shared/comparegamebackends).
2. Choose **Report a vulnerability** to open a private security advisory.

This keeps the details private until any fix is in place. We will review the report and respond as time allows.

## Scope

In scope: the application code and build configuration in this repository, and the deployed site.

Out of scope: the third-party platforms compared on the site. For a security issue in PlayFab, AWS GameLift, Unity Gaming Services, or any other platform listed here, please contact that vendor directly.

To report a factual error in the comparison data (not a security issue), open a [data-correction issue](https://github.com/metaplay-shared/comparegamebackends/issues/new/choose) instead.
